# Sales Performance Dashboard (Power BI)

## Project Overview
This project presents a Sales Performance Dashboard built using Power BI to analyze sales trends, product performance, customer behavior, and regional market distribution.

The dashboard enables businesses to monitor key sales metrics and identify growth opportunities.

---

## Key Features
- Total Sales, Quantity, and Profit KPIs
- Sales trend analysis across years
- Market-wise sales distribution
- Product category performance
- Top customers analysis
- Interactive filters for segment, category, and country

---

## Tools Used
Power BI  
DAX  
Excel  
Data Visualization  

---

## Key Insights
- Certain markets contribute significantly higher sales volume
- Top customers generate a large share of revenue
- Some product categories outperform others consistently
- Sales and profit trends show steady growth over the years
